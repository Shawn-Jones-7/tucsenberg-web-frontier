# 外贸企业信息文档模板（供 LLM/设计/开发共同使用）

这份文档的目标是把“企业真实信息 + 业务战略 + 网站呈现策略”固化成**一个可被 LLM 直接消费的单一真源**（Single Source of Truth），用于后续按 `docs/frontend-llm-workflow.md` 的工作流稳定产出生产级页面与内容。

> 使用方法（推荐）  
> 1) 先把已有资料（公司介绍、产品目录、证书、FAQ、联系方式、成交案例等）尽量填满。  
> 2) 再让 LLM（例如 ChatGPT/Claude）补全“战略/叙事/呈现策略”，但必须标注哪些内容来自事实、哪些是建议。  
> 3) 最后由你（业务负责人）确认/定稿：未确认的信息不得进入网站对外文案。
