# Project Overview
- Name: B2B Web Template
- Goal: Provide a modern B2B enterprise website baseline with English-Chinese bilingual support, theme switching, performance and observability capabilities, meeting enterprise-grade quality and security requirements.
- 核心技术：Next.js 16（App Router）、React 19、TypeScript 5.9、Tailwind CSS 4、MDX 内容、next-intl 国际化。
- 关键特性：多语言/主题、性能基线与回归检测、Sentry 集成、安全与架构检查脚本、Playwright E2E、Vitest 单元测试、质量分析脚本。
