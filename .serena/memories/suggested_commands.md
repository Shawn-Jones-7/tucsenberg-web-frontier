# 常用命令
- 安装依赖：`pnpm install`
- 启动开发：`pnpm dev --turbo`
- 构建与启动：`pnpm build && pnpm start`
- 单元测试：`pnpm test`（覆盖率 `pnpm test:coverage`）
- 类型检查：`pnpm type-check`；测试类型：`pnpm type-check:tests`
- Lint/格式：`pnpm lint:check`、`pnpm lint:fix`、`pnpm format:check`、`pnpm format:write`
- 质量门禁：`pnpm quality:gate`、`pnpm quality:monitor`
- 架构/安全：`pnpm arch:check`、`pnpm security:audit`
- E2E 测试：`pnpm test:e2e`
- 配置一致性：`pnpm config:check`
- 依赖体积：`pnpm size:check`
